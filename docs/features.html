Tip

The following features are available for multiple spatial scales:

  • feat_topo_eumohp_<feature name>(1, 2, 3, 4, 5)
  • feat_topo_amatulli_<feature name>(1km, 5km, 10km, 50km, 100km)<suffix>

So please be aware of that, when subsetting features!
